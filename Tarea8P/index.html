Colores multinavegador

Hay mas de 160 colores con nombre que son compatibles con todos los navegadores modernos. Estos nombres se puden usar directamente en css para aplicar colores a elementos sin especificar valores hexadecimales, egb o hsl.

Por ejemplo:

  1. Azul
  2. Rosa
  3. Naranja
  4. Violeta
  5. Amarillo

Colores de un borde

Para que a un elemento HTML se le puede aplicar color a su borde son necesarias 2 propiedades: border style y border-color
