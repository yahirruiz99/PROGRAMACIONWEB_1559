Lenguaje de Marcado de Hipertexto(HTML)

¡Bienvenidos! Hoy estaremos practicando algunos conceptoa y elementos más importantes vistos en hasta el momento en la materia de Programación Web I, además veremos el uso de etiquetas semánticas en la estructura de un sitio web.

Conceptoa principales de HTML

HTML

Lenguaje de Marcado de Hipertexto, que se utiliza para hacer o esructurar una página web.

Elemento HTML

Es la unidad básica de estructuray contenido de una página web, esta compuesta de:

  1. Etiqueta de apertura
  2. Pares atributo-valor
  3. Contenido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta que contiene metadatos, es la forma en la que una página web se comunica con el navegador, funciona para el SEO, se puede enlazar archivos CSS ó Scripts.

Etiqueta <body>

Contiene todo el contenido HTML

Contenedores Semánticos

<header> Etiqueta utilizada para el contenido introductorio de una página web puede contener el logo, el menú de navegación principal, el nombre de la empresa, etc.
<navr> Etiqueta utilizada para el menú de navegación principal, suele contener enlaces hacia otras páginas relacionada o navegar por la página principal, no debe haber mas de una dentro de la página.
<main> Etiqueta única de una página web, cuyo proposito es resaltar nuestro objetivo o mensaje principal.
<section> Es una etiqueta que se intenta que sustituya a div, su uso es para dividir la página en secciones y agrupar contenido relacionado.
<article> Es una etiqueta que se utiliza para agrupar contenido áutonomo, que si se separa del contenido principal, sigue teniendo coherencia, normalmente se compone de un encabeza y un contenido
<footer> Es una etiqueta que normalmene se encuentra al final de una página web y funciona como cierre, a menudo contiene información de contacto, derechos de autor o algun otro dato complementario.

Etiquetas principales HTML

Etiqueta <a>

Es un elemento que nos va a permitir insertar un enlace a otra página:

  • Navegar a páginas externas
  • Navegar a páginas internas
  • Navegar a secciones dentro de nuestra misma página(Enlaces ANCLA)

Algunos valores especificos para el atributo href son:

    href="https://wa.me/5577825278
    Abre un chat del número especificado en Whatsap
    href="mailto:example@.com"
    Abre la aplicación de correo con el destinatario especificado
